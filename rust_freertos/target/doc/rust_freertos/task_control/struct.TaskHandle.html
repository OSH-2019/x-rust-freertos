pub struct TaskHandle(_);

Type by which tasks are referenced. For example, a call to xTaskCreate returns (via a pointer parameter) an TaskHandle_t variable that can then be used as a parameter to vTaskDelete to delete the task. Since multiple TaskHandles may refer to and own a same TCB at a time, we wrapped TCB within a tuple struct using Arc<RwLock<_>>
